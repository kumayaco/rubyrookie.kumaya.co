## Full List of Comparison

 Comparison | Meaning 
 ---------- | ------- 
 `x > y` | Greater than, 大於
 `x < y` | Less than, 小於 
 `x == y` | Equal to, 等於
 `x >= y` | Greater than *or* equal to, 大於或等於
 `x <= y` | Less than *or* equal to, 小於或等於
 `x <=> y` | Comparison; return `0` if `x` and `y` are equal, `1` if `x` is higher, and `-1` if `y` is higher
 `x != y` | Not equal to, 不等於
